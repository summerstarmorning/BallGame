# 版本更新记录

## [v1.6.0] - 2026-04-13
### 新增 (Added)
- **屏幕边缘粒子反射特效**：当小球碰撞左/右/上边界时会触发粒子喷发，粒子可在四周边缘继续反弹并衰减消失。
- **粒子生命周期管理**：新增粒子寿命、透明度渐隐与数量上限控制，避免长期运行后粒子无限累积导致性能下降。

### 修改 (Changed)
- 在 `Game` 主循环中接入粒子更新与渲染流程，确保特效在 `PLAYING/PAUSED/GAMEOVER` 场景中可见且行为一致。

---

## [v1.5.0] - 2026-04-03
### 新增 (Added)
- **关卡选择**：在主菜单界面新增了两个按钮 (`Level 1` 和 `Level 2`)，允许玩家在开始游戏前指定想要挑战的关卡。
- **无敌测试模式**：在游戏内按 `Z` 键可以切换进入调试无敌模式 `[DEBUG: INVINCIBLE]`。该模式下小球掉落屏幕不会扣除生命值且生命值显示为 `999`。
- **用户配置持久化**：玩家设定的 `Theme` 亮暗模式以及小球、挡板和砖块的最后一次选定颜色将被暂存至 `user_prefs.json` 当中，重新启动游戏时能够实现无缝恢复。
- **自动化测试集成**：新增了分离的测试执行文件以及对应的单元测试以验证小球对砖块的物理碰撞算法效果是否正确。

### 修改 (Changed)
- **砖块布局更新**：重构了 `InitBricks()` 初始化砖块的渲染逻辑。现在默认支持行/列的自由配平，生成了更多的堆叠砖块（4列8行网格），并增加了它们之间的纵向缝隙使得排布视野更好。

---

## [v1.4.0] - 2026-04-03
### 新增 (Added)
- 新增浅色与深色主题模式 (Light / Dark mode)，可在主界面菜单中点击按钮一键切换全局颜色方案。
- 在主菜单界面 `MENU` 状态增加了实体颜色配置模块：通过点击专门的按钮即可方便地循环切换小球、挡板和砖块的显示颜色。

### 优化 (Optimized)
- 优化与美化了整体 UI 设计，放弃生硬的直角矩形，改为柔和的圆角矩形渲染 (挡板、砖块、配置按钮等)。
- 为按钮和砖块加入了部分半透明的描边使其更具立体质感。

---

## [v1.3.0] - 2026-04-03
### 新增 (Added)
- 引入了完整的状态机，目前支持 `MENU`（主菜单菜单）、`PLAYING`（游戏中）、`PAUSED`（暂停）、`GAMEOVER`（游戏结束）四种状态。
- 新增 `config.json` 配置文件读取，使得游戏参数（生命值维度、球的移动速度及坐标、挡板相关参数）可配化并支持带有注释的 JSON 解析。
- 在游戏界面右上角增加了一个可见的、可点击的暂停/继续图形化按钮。

### 修改 (Changed)
- 简化并优化了挡板和球的碰撞逻辑，取消了“根据击球位置决定球速大小”的设计，修改为仅叠加挡板自身移动时的切向速度。
- 配置文件中增加了明确的中文注释功能，方便参数调整与学习。

---

## [v1.2.0] - 2026-04-02
### 新增 (Added)
- 新增了 `Game` 类，将主逻辑（状态、物理更新、渲染绘制）封装到 `Game.h` 和 `Game.cpp` 文件中，将 `main` 函数进行了彻底的解耦与重构。

### 修改 (Changed)
- 整体游戏窗口扩大，从 `800x600` 放大为 `1024x768`。
- 调整了游戏实体的视觉尺寸，以适应新屏幕比例：球的半径变大为 15.0，挡板尺寸变为 150x25，砖块拉长为 120x40。
- 优化了物理手感：大幅削弱了打擦边球和甩拍子时的增速效应（击球偏移系数 `HIT_OFFSET_FACTOR` 降至 4.0，挡板滑动影响 `PADDLE_INFLUENCE` 降至 0.2，并降低了最高速度上限），现在球速更加平缓可控。
- 将掉命后重新发球的坐标重置为了大屏幕的正中心。

---

## [v1.1.0] - 2026-04-02
### 新增 (Added)
- 增加了生命值 (Lives) 系统，初始3条命。
- 增加了得分 (Score) 系统，击碎每个活跃砖块得100分。
- 增加了游戏胜利与游戏失败的状态切换和文字UI提示。
- 加上了 `docs` 文档目录，用于记录更新与心得。

### 修改 (Changed)
- 移除了屏幕底部的碰撞墙，球掉出底沿会扣除生命值并重置球心位置。
- 优化了球与挡板的碰撞反弹手感。现在的出射角度结合了“击球偏移水平增量”和“挡板移动速度的影响”，并且防止了球卡在挡板内反复翻转的问题。

---

## [v1.0.0] - 2026-04-02
### 新增 (Added)
- 游戏的基础框架建立：包含 raylib 窗口初始化，球、挡板、砖块三个独立基础类的实现和渲染。
- 能够编译执行和移动挡板的初期版本。

### 修复 (Fixed)
- 源码包含 `<windows.h>` 会遇到函数重名冲突（例如 `CloseWindow`），已修复为使用 `system("chcp 65001")` 代替来处理控制台输出乱码。
- 解决了所有的 C++ 源文件在 MSVC 编译器下（代码页 936）的中文注释乱码或告警问题，全部统一转化为了 UTF-8 编码。
- 修复了本地 git 往 GitHub 同步时出现 Connection was reset 的错误（现已成功改造成 SSH 连接）。
