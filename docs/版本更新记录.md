# 版本更新记录

## [v1.2.0] - 2026-04-02
### 新增 (Added)
- 新增了 `Game` 类，将主逻辑（状态、物理更新、渲染绘制）封装到 `Game.h` 和 `Game.cpp` 文件中，将 `main` 函数进行了彻底的解耦与重构。

### 修改 (Changed)
- 整体游戏窗口扩大，从 `800x600` 放大为 `1024x768`。
- 调整了游戏实体的视觉尺寸，以适应新屏幕比例：球的半径变大为 15.0，挡板尺寸变为 150x25，砖块拉长为 120x40。
- 优化了物理手感：大幅削弱了打擦边球和甩拍子时的增速效应（击球偏移系数 `HIT_OFFSET_FACTOR` 降至 4.0，挡板滑动影响 `PADDLE_INFLUENCE` 降至 0.2，并降低了最高速度上限），现在球速更加平缓可控。
- 将掉命后重新发球的坐标重置为了大屏幕的正中心。

---

## [v1.1.0] - 2026-04-02
### 新增 (Added)
- 增加了生命值 (Lives) 系统，初始3条命。
- 增加了得分 (Score) 系统，击碎每个活跃砖块得100分。
- 增加了游戏胜利与游戏失败的状态切换和文字UI提示。
- 加上了 `docs` 文档目录，用于记录更新与心得。

### 修改 (Changed)
- 移除了屏幕底部的碰撞墙，球掉出底沿会扣除生命值并重置球心位置。
- 优化了球与挡板的碰撞反弹手感。现在的出射角度结合了“击球偏移水平增量”和“挡板移动速度的影响”，并且防止了球卡在挡板内反复翻转的问题。

---

## [v1.0.0] - 2026-04-02
### 新增 (Added)
- 游戏的基础框架建立：包含 raylib 窗口初始化，球、挡板、砖块三个独立基础类的实现和渲染。
- 能够编译执行和移动挡板的初期版本。

### 修复 (Fixed)
- 源码包含 `<windows.h>` 会遇到函数重名冲突（例如 `CloseWindow`），已修复为使用 `system("chcp 65001")` 代替来处理控制台输出乱码。
- 解决了所有的 C++ 源文件在 MSVC 编译器下（代码页 936）的中文注释乱码或告警问题，全部统一转化为了 UTF-8 编码。
- 修复了本地 git 往 GitHub 同步时出现 Connection was reset 的错误（现已成功改造成 SSH 连接）。
