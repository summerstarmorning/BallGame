# 打砖块游戏 (Breakout Clone)

这是一个使用 C++ 和 `raylib` 库创建的简单二维“打砖块”游戏项目。

## 项目状态

这是一个正在开发中的项目。

### 已实现功能

*   **窗口创建**: 创建一个 800x600 的游戏窗口。
*   **游戏对象**:
    *   一个可以自动移动的 **小球 (Ball)**。
    *   一个玩家可以通过键盘左右控制的 **挡板 (Paddle)**。
    *   一排静态的 **砖块 (Bricks)**。
*   **基本物理**:
    *   小球会在窗口的四个边界处反弹。
*   **渲染**:
    *   所有游戏对象（球、挡板、砖块）都会被绘制在屏幕上。

### 待实现功能

*   **碰撞检测**:
    *   小球与挡板的碰撞。
    *   小球与砖块的碰撞。
*   **游戏逻辑**:
    *   击碎砖块后的逻辑（例如，砖块消失）。
    *   游戏失败条件（例如，小球掉落到屏幕底部）。
    *   游戏胜利条件（例如，所有砖块都被击碎）。
*   **计分系统**。
*   **多关卡**。

## 如何构建和运行

本项目使用 CMake 进行构建。

### 依赖

*   `raylib` 游戏库
*   C++ 编译器 (支持 C++11 或更高版本)
*   CMake (版本 3.10 或更高)

### 构建步骤

1.  **安装依赖**: 确保你已经安装了 `raylib` 和 `CMake`。
2.  **创建构建目录**:
    ```bash
    mkdir build
    cd build
    ```
3.  **生成构建文件**:
    ```bash
    cmake ..
    ```
4.  **编译项目**:
    *   在 Windows (Visual Studio) 上:
        ```bash
        cmake --build .
        ```
    *   在 Linux/macOS 上:
        ```bash
        make
        ```

### 运行游戏

编译成功后，可执行文件 `breakout_week2` 会生成在 `build` 目录中。
